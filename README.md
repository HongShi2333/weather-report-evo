# 🌦️ 红石天气 (Redstone Weather)

一个基于 **极简主义** 和 **玻璃拟态 (Glassmorphism)** 设计的高颜值、响应式单文件天气仪表板。

## ✨ 核心特性

* **超薄玻璃视效**：采用极致透明的磨砂玻璃设计，在确保文字清晰可读的同时，最大程度还原背景壁纸的美感。
* **智能动态背景**：
* 自动识别设备类型（移动端/桌面端）并加载对应的适配比例壁纸。
* 内置双层平滑切换动画，每 25 秒自动更换风景，提供丝滑的视觉过渡。


* **响应式 HUD 布局**：
* 顶部信息条（HUD）在移动端会自动精简标签，确保在窄屏设备上时间与定位信息不重叠、不被遮挡。


* **全方位天气数据**：
* 实时监测：温度、体感、湿度、风速、气压、能见度。
* 生活参考：日出日落、月相、生活建议。
* 趋势预报：未来 48 小时及两日详细预报，包含降水概率、平均湿度、风力等细节。


* **灵动交互**：
* 具有“呼吸感”的加载层，在数据同步完成后以缩放淡出的方式自然退出。
* 平滑的锚点跳转导航。



## 🚀 快速开始

该项目是一个 **单文件 (Static HTML)** 应用，无需任何复杂的安装过程。

1. **下载代码**：获取 `index.html` 文件。
2. **准备壁纸**：在 `index.html` 同级目录下创建以下文件夹结构并放入你的 `.jpg` 图片：
```text
├── desktop/         # 存放电脑端壁纸，命名如 d-background1.jpg, d-background2.jpg...
├── mobile/          # 存放手机端壁纸，命名如 m-background1.jpg, m-background2.jpg...
└── index.html

```


3. **直接运行**：使用任意现代浏览器（Chrome, Edge, Safari, Firefox）打开 `index.html` 即可。

## 🛠️ 技术栈

* **HTML5**：语义化布局。
* **CSS3**：
* `backdrop-filter` 实现磨砂玻璃。
* `CSS Variables` 方便一键调色。
* `Grid & Flexbox` 实现复杂的响应式排版。


* **Vanilla JavaScript**：纯原生 JS 处理 API 调用、背景切换逻辑和时间显示，无需任何重量级框架。
* **API 支持**：
* 定位：基于 IP 的位置解析服务。
* 天气：红石镜像天气源 (Hongshi Weather API)。



## ⚙️ 进阶自定义

您可以直接在 `index.html` 的 `:root` 中快速调整视觉风格：

```css
:root {
  --glass: rgba(255, 255, 255, 0.04); /* 调整卡片透明度 */
  --radius-xl: 28px;                  /* 调整圆角大小 */
  /* 其他变量... */
}

```
